LIFTO - Laboratórios Inteligentes do IFTO

O que é o LIFTO?

O LIFTO revitaliza computadores obsoletos com software livre, criando laboratórios digitais completos para escolas públicas. Uma solução que une sustentabilidade, economia e inclusão digital.

Criação

Criado pelo G-REDES/IFTO, o LIFTO revitaliza computadores e desenvolve ferramentas livres para escolas públicas, garantindo aprendizado digital acessível a todos.

Princípios

  • Sustentabilidade Tecnológica:Revitalização de computadores obsoletos com software livre, reduzindo lixo eletrônico e custos.
  • Acessibilidade e Inclusão Digital : Garantir que escolas públicas tenham laboratórios funcionais, mesmo com equipamentos antigos.
  • Educação Pública de Qualidade : Oferecer ambientes digitais completos para apoio pedagógico.
  • Autossuficiência Tecnológica: Sistemas como o Bot de Alerta e Instalador de Programas para manutenção simplificada.

COMO USAR ?

Introdução ao Sistema

O LIFTO foi desenvolvido para ser intuitivo e acessível a todos os usuários, independentemente do nível de conhecimento técnico. Este guia explica passo a passo como utilizar as principais ferramentas do sistema.

1

Acesso inicial

Abra seu navegador(Chrome, Firefox, Edge, etc.) e digite o link do LIFTO(ex:http://lifto.ifto.edu.br)
A página inicial carregará automaticamente, exibindo todas as opções disponíveis.

2

Seleção de Ferramentas

Na seção "Aplicações", clique no botão correspondente à ferramenta que deseja usar (ex: Instalador de Programas, Bot de Alerta, etc.).
Cada aplicação abrirá em uma nova janela ou guia, dependendo da sua configuração de navegador.

3

Uso das Funcionalidades

Siga as instruções específicas de cada ferramenta (elas serão exibidas na tela assim que você acessá-las).
Exemplo: No Instalador de Programas, basta selecionar os softwares desejados e clicar em "Instalar".

4

Logout e Segurança

Sempre que terminar de usar o sistema, feche as abas do navegador ou clique em "Sair" (se disponível) para garantir a segurança dos dados.

Recomendações para manutenção e atualização

✅ Para usuários comuns (professores/gestores):

  • Verifique periodicamente se as ferramentas estão respondendo corretamente.
  • Em caso de erro, recarregue a página (tecla F5 ou ícone de atualização no navegador).
  • Limpe o cache do navegador a cada 2 meses (Configurações > Privacidade > Limpar dados de navegação).

✅ Para técnicos de TI:

  • Monitore o servidor do LIFTO (uso de CPU/memória) semanalmente.
  • Reinicie o serviço mensalmente para evitar lentidão (sudo systemctl restart lifto em Linux).
  • Faça backups dos arquivos de configuração (ex: /etc/lifto/config.json) antes de atualizações.

Atualizações do Sistema

Atualizações automáticas:

O LIFTO verifica atualizações ao ser iniciado. Se uma nova versão estiver disponível, um aviso aparecerá no canto inferior direito da tela.

Atualizações manuais (para administradores):

  • Acesse o Painel de Controle do LIFTO (disponível para técnicos).
  • Navegue até "Atualizações" > "Verificar atualizações".
  • Siga o assistente para baixar e instalar.
  • Reinicie o servidor após a instalação.

Recomendação:

Agende atualizações para horários fora do expediente (ex: finais de semana) para evitar interrupções.

3. network_check.py - Verifica conexão das estações 
import os
stations = ["lab01", "lab02", "lab03"] # Lista de IPs/hostnames
for s in stations:
    response = os.system("ping -c 1 " + s)
    print(f"{s}: {'Online' if response == 0 else 'Offline'}")
